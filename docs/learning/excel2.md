# Excelの基礎2
今回は、Excelでよく使われる"関数"という機能を使ってみましょう。

## 関数とは
Excelは表の作成を便利にするとともに、計算も簡単にしてくれる機能が付いています。それが関数です。
合計、平均、数値の比較など、対象となる数がどれだけ大量にあっても一瞬で計算してくれて、データを扱う上では欠かせない技術になるのでしっかり身につけてください。
